# 修正メモ

## 対応予定

1. **配信中に削除されたコメントがアプリ側に残る**  
   配信で配信中に削除したコメントが、アプリ側の表示として残ってしまう挙動を修正する。

2. **絵文字コメントが英数字の文字列でしか表示されない**  
   ライブ配信で絵文字のコメントをもらったときに、アプリ側では英数字の文字列でしか表示されない挙動を修正する。

3. **独自絵文字の追加**  
   独自絵文字に対応する（追加する）。

4. **吹き出しのしっぽ付近でコメントテキストが消える**  
   吹き出しのダイヤ（しっぽ）の部分でコメントのテキストが消えてしまう挙動を修正する。
